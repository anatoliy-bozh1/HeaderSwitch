# HeaderSwitch (v1.0.0)

**Разработчик:** Божко Анатолий, группа M3113  
**IDE:** Visual Studio Code  

## Описание

HeaderSwitch — это расширение для **Visual Studio Code**, которое позволяет быстро переключаться между C++ исходными файлами (.cpp) и заголовочными файлами (.h/.hpp).  

Основные возможности:  
- Быстрое открытие парного файла (header <-> source)
- Автоматическое создание нового файла, если парный не найден  
- Новый файл создается в правильной папке (`src` для `.cpp`, `include` для `.h/.hpp`)  

## Установка

1. Откройте **Visual Studio Code**  
2. Перейдите в меню **Extensions** (Расширения) или нажмите `Ctrl+Shift+X`  
3. Найдите расширение `HeaderSwitch` по названию или установите локально через `Install from VSIX`  
4. После установки команду `Switch Header/Source` можно использовать через `Ctrl+Shift+P` -> поиск команды `HeaderSwitch: Switch Header/Source` 

## Использование

1. Откройте файл `.cpp`, `.h` или `.hpp`  
2. Вызовите команду **Switch Header/Source**  
3. Если парный файл найден, он откроется  
4. Если парный файл не найден:  
   - Выберите расширение нового файла  
   - Новый файл будет создан в соответствующей папке (`src` или `include`) с базовым шаблоном содержимого  

## Архитектура проекта / как работает расширение

Немного о том, как работает плагин:
- Плагин слушает команду headerswitch.switch
- Определяет открытый файл — исходник или заголовок
- Ищет парный файл в текущей папке, include и src
- Если парный найден — открывает, если нет — предлагает создать новый файл с базовым шаблоном

## Возможности / ограничения расширения

### Возможности
 
- Быстрое переключение между header и source
- Автоматическое создание нового файла в правильной папке

### Ограничения

- Поддерживает только .cpp, .h и .hpp
- Новый файл создается с базовым шаблоном, без сложной генерации содержимого
