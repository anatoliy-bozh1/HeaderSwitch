# HeaderSwitch (v1.0.0)

**Автор и разработчик:** Божко Анатолий, студент ИТМО, группа M3113  
**IDE:** Visual Studio Code (плагин работает с файлами C++ `.cpp`, `.h`, `.hpp`)

## Описание проекта

**HeaderSwitch** — расширение для **Visual Studio Code**, которое позволяет быстро переключаться между исходными C++ файлами (`.cpp`) и заголовочными файлами (`.h` / `.hpp`).  

Основные возможности:  
- Мгновенное открытие парного файла (`header` <-> `source`)  
- Автоматическое создание нового файла, если парный не найден  
- Новый файл создается в правильной папке: `src` для `.cpp`, `include` для заголовков  

## Установка

1. Откройте **Visual Studio Code**  
2. Перейдите в меню **Extensions** (Расширения) или нажмите `Ctrl+Shift+X`  
3. Выберите `Install from VSIX…` и укажите файл `HeaderSwitch-1.0.0.vsix`, который был сгенерирован с помощью команды `vsce package`  
4. После установки откройте командную палитру `Ctrl+Shift+P` и выполните команду `HeaderSwitch: Switch Header/Source`  

## Использование

1. Откройте файл C++ с расширением `.cpp`, `.h` или `.hpp`  
2. Вызовите команду `HeaderSwitch: Switch Header/Source` через командную палитру `Ctrl+Shift+P`  
3. Если парный файл найден, он откроется; если нет — выберите расширение нового файла, который будет создан с базовым шаблоном в правильной папке (`src` для `.cpp`, `include` для заголовков)  

## Архитектура проекта / как работает расширение

HeaderSwitch работает следующим образом:  
- Слушает команду `headerswitch.switch`  
- Определяет тип текущего открытого файла — исходный `.cpp` или заголовочный `.h`/`.hpp`  
- Ищет парный файл в текущей папке, `include` и `src`  
- Если парный найден, открывает его; если нет — предлагает создать новый файл с базовым шаблоном содержимого  

## Возможности и ограничения расширения

### Возможности

- Быстрое переключение между заголовочными и исходными файлами (header <-> source)  
- Автоматическое создание нового файла в правильной папке (`src` для `.cpp`, `include` для `.h`/`.hpp`)  

### Ограничения

- Поддерживаются только файлы с расширениями `.cpp`, `.h` и `.hpp`  
- Новый файл создается с базовым шаблоном содержимого, без сложной генерации кода
